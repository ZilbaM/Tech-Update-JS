Apexcharts JS

Apexcharts JS is a JavaScript library that allows you to create charts of multiple types to create a visual indicators of data on your web page.

Set up

There are 2 ways to install ApexchartsJS on your website. The first is by installing via npm.

npm install apexcharts --save

Usage:

import ApexCharts from 'apexcharts'

The other way to install it is by including it directly into your HTML:

<script src="https://cdn.jsdelivr.net/npm/apexcharts"></script>

Usage

Creating a chart is really easy. First you create an empty div tag with an id in your HTML.

<div id="chart"></div>

Then, in your Javascript file, you write your charts parameters :

Code

var options = {
    chart: {
      type: 'line'
    },
    series: [{
      name: 'sales',
      data: [30,40,35,50,49,60,70,91,125]
    }],
    xaxis: {
      categories: [1991,1992,1993,1994,1995,1996,1997, 1998,1999]
    }
  }
  
  var chart = new ApexCharts(document.querySelector("#chart"), options);
  
  chart.render();
  • chart : it only contains the type of chart you want. See below for more chart types.
  • series : The series is a set of data. You may have single or multiple data series.
  • xaxis : In xaxis, you put the categories or values you want on the horizontal axis.

Parameters

Chart type :

You can easily change the chart type just by changing the type argument in the script :

Code

chart: {
    type: 'bar'
}

Ouput

Paired Values

You can have single or multiple data series.

Code

series: [{
  name: 'sales',
  data: [30,40,35,50,49,60,70,91,125]
},{
  name: 'viewers',
  data: [25,36,90,60,65,55,68,100,115]
}];

There are multiple ways to add other series to your chart. However, they each require technical syntax. Make sure to check the official documentation of Apexcharts JS before you do anything, to be certain you know the right way to do what you want.
