Child Nodes

ChildNodes is a read-only property in JavaScript that returns a shit ton of infos about the HTML childs of the targeted tag.

It basically allow you to get every JS and HTML related properties of the tag.

How to use it

The proper way to use ChildNodes is to first target an element with its class, its id or a querySelector.

Then, use an element.nodesChild to put the data in a variable

Exemple

var parent = document.getElementsByClassName('parent')[0];
//Or var parent = document.querySelector('.parent');
//After the element is selected, we assign its childNodes to a new variable
var nodes = parent.childNodes;
console.log(nodes);

Output

As you can see above, the console.log returns an array with 3 entries : 2 texts and the child of the targeted.

The 2 texts are corresponding to the empty space between the <div> tag and the <p> tag.

When we unroll the second element of the array, we can see that it is an array too with a lot of entries

A few among them are :


  • childNodes: You can see the childNodes of the child and so on
  • classList: an array with the classes of the element (it also contains a list of action you can do to the classList (Ex : toggle, add, remove, replace...))
  • A list of every event and if there is an action linked to it with an addEventListener
  • outerHTML: a string with the entirety of the tag (Ex : "<p class=\"child\">Hello World !</p>")
  • outerText: a string with the text inside the html tag (Ex: "Hello World !")
